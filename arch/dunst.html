Dunst Notifier

Dunst runs on a fairly default config. I adjusted the position of it, so it matches the edges of my other windows in the upper right corner. Furthermore I altered the colors to match my colorscheme.

~/.config/dunst/dunstrc

1: # See dunst(5) for all configuration options
2:
3: [global]
4: ### Display ###
5:
6: # Which monitor should the notifications be displayed on.
7: monitor = 1
8:
9: # Display notification on focused monitor. Possible: mouse, keyboard, none
10: follow = none
